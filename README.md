# cybersecurity-task-4
This document presents the EncryptEdge Labs Cybersecurity Analyst Internship Task Report (Task No: 04). It outlines foundational cybersecurity concepts, key cyber threats (malware, phishing, DDoS, insider threats), and the critical roles of a cybersecurity analyst, including monitoring, vulnerability management, and incident response. The report details practical application through an internal penetration test conducted on EncryptEdge Labs' network, highlighting identified and exploited vulnerabilities, alongside hands-on experience gained from TryHackMe labs. It concludes with reflections on developed understanding of security operations and recommendations for continued learning.
